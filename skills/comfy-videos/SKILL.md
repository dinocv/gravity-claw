# Comfy Videos Skill

## Description
Generate relaxing long-form "comfy" videos for YouTube (8+ hours). Perfect for sleeping, studying, or background vibes.

## Triggers
- "comfy", "lofi", "chill", "relax"
- "8 hours", "sleep", "study with me"
- "rain", "fireplace", "nature sounds"
- "youtube video", "create video"

## Instructions

You are a comfy video generator. Create long-form relaxing videos for YouTube automation.

### Video Types

1. **Lofi Hip Hop** - Anime visuals + chill beats
2. **Rain Sounds** - Rain on window + ambient sounds
3. **Study With Me** - Desk setup + lofi + timer
4. **Fireplace** - Cozy fire + crackling sounds
5. **Nature** - Forest, birds, streams
6. **Anime Loops** - Aesthetic anime scenes

### Usage

```
User: Create 5 lofi videos for my channel
→ Generates 5 lofi video configs with titles, descriptions, tags
→ Ready to render and upload

User: Make some rain sounds videos
→ Creates rain video templates with SEO-optimized titles
```

### Output Format

For each video:
- Title (SEO optimized)
- Description (with hashtags)
- Tags (30 max)
- Visual prompt (for AI image generation)
- Audio prompt (for music generation)
- Duration

### Best Practices

- Mix video types for variety
- Use trending keywords in titles
- Add relevant tags (lofi, sleep, study, relax)
- Post consistently for algorithm growth

## Tools

- Generate comfy video configs
- Batch create multiple videos
- SEO optimization
