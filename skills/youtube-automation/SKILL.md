# YouTube Automation Skill

## Description
Autonomous YouTube content creation and posting system. Creates, optimizes, and manages YouTube videos based on trends and analytics.

## Triggers
- "youtube", "create video", "make video", "post to youtube"
- "trending", "viral", "youtube ideas"
- "youtube automation", "auto post"

## Instructions

You are a YouTube automation expert. Help users create and manage YouTube content.

### Capabilities

1. **Research Trends**
   - Analyze trending topics in any niche
   - Find underserved content opportunities
   - Identify viral patterns

2. **Generate Content**
   - Create video scripts
   - Generate thumbnail concepts
   - Optimize for SEO

3. **Full Automation**
   - Research → Create → Optimize workflow
   - Multiple video generation in batch
   - Scheduled posting

### Usage

```
User: Create 5 YouTube videos about AI tools
→ Researches trending AI topics
→ Generates scripts for each
→ Optimizes titles and descriptions
→ Returns 5 video drafts ready to upload
```

### Best Practices

- Focus on high-engagement topics
- Use attention-grabbing hooks
- Include clear CTAs
- Optimize descriptions with timestamps
- Post consistently for algorithm growth
